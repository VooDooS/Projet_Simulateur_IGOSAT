Description détaillée

Classe abstraite pour les connecteurs des modules.

Les sockets modélisent les connections entre les modules. Un connecteur peut-être soit un connecteur d'entrée, InSocket, soit un connecteur de sortie, OutSOcket. Les connecteurs sont reliés entre-eux via les objets Connexion.
